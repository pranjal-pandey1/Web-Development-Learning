The <body> element typically contains structural elements such as:

  • <header> — document or site header (logo, navigation).
  • <main> the main content area for the current page.
    • <section> — Divides the wesite into sections.
    • <article> — Is used when the content makes sense by itself and can be shared or reused independently (like a single story, post, or entry).
    • <aside> — Defines content that is indirectly related to the main content — like a sidebar, extra information, or complementary notes.
  • <footer> — footer information (copyright, links).
