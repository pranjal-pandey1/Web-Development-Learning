Simple Explanation of Block Level and Inline Elements

Block-level elements (e.g., div, p, h1, ul) create structure. Each starts on a new line and by default stretches the full width of its container.

Inline elements (e.g., span, a, strong, img) sit inside lines of text. They do not start a new line and only take as much width as their content. 

Note: Inline elements cannot contain block elements (e.g., a span should not wrap a div). CSS can change behavior: display: inline-block or display: block alter defaults.Replaced inline elements (like img) behave a bit differently: you can set width/height.
