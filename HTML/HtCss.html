About Css

Css is used to style the HTML elements on the web page. It allows you to control the layout, colors, fonts, and overall appearance of your website. With CSS, you can create visually appealing and responsive designs that enhance the user experience.

Css can be styled in 3 ways:

  • Inline CSS: Applied directly within an HTML element using the "style" attribute. Example: This text uses inline CSS. 
    • <span style = "color: red; font-weight: bold;">This text uses inline Css</span>
  • Internal CSS: Defined within a <style> tag in the <head> section of an HTML document. 
    <head>
      <style>
        .internal-example {
          color: green;
          font-weight: bold;
        }
      </style>
    </head>
    <body>
      <p class="internal-example">This text uses internal CSS.</p>
    </body>
  • External CSS: Linked to an external stylesheet file using the <link> tag in the <head> section. 
    <head>
      <link rel="stylesheet" href="styles.css">
    </head>
    <!-- styles.css -->
    .external-example {
      color: purple;
      font-weight: bold;
    }
    <body>
      <p class="external-example">This text uses external CSS.</p>
    </body>

Selectors in Css

Selectors are used to target HTML elements and apply styles to them. There are several types of selectors in CSS, including:
  1. Universal Selector: Selects all Element in a Page and apply style to whole Page 
    
        *{
          margin: 0;
          padding: 0;
          box-sizing: border-box; 
        }
  2. Element Selector: Select all element of a speacific HTML tag 
    
        p {
          color: blue;
        }
  3. Class Selector: Select all element with a speacific class attribute 
    
        .classname {
          font-size: 20px;
        }
  4. ID Selector: Select a speacific element with a speacific id attribute 
    
        #uniqueID {
          background-color: yellow;
        }
  5. Descendant Selector: Select all element that are nested within a speacific element 
    
        div p {
          color: green;
        }
  6. Grouping Selector: Apply same style to multiple element 
    
        h1, h2, h3 {
          font-family: Arial, sans-serif;
        }

Color System

In CSS, colors can be specified using various methods, including:

  • Named Colors: e.g., red, blue, green.
  • Hexadecimal (Hex) Codes: e.g., #FF5733, #00FF00.
  • RGB (Red, Green, Blue): e.g., rgb(255, 0, 0), rgb(0, 255, 0).
  • RGBA (Red, Green, Blue, Alpha): e.g., rgba(255, 0, 0, 0.5) for semi-transparent colors.

Units In Css

CSS supports various units for specifying lengths and sizes, including:

  • Absolute Units: px (pixels), cm (centimeters), mm (millimeters), in (inches), pt (points), pc (picas).
  • Relative Units: em (relative to the font-size of the element), rem (relative to the font-size of the root element), % (percentage of the parent element's size), vw (viewport width), vh (viewport height).
Pixels (px) : A pixel is a fixed unit that represents a single dot on the screen. It is commonly used for font sizes, margins, and padding.
