About List and Its Types

A list in html is a way to organize and display related items together.

Types of Lists in HTML

1. Ordered List (<ol>)

An ordered list is used when the sequence of items matters. It displays items in a numbered format.

Ordered List Type can be Numeric by default,roman numbers and alphabatical

Example of Ordered List:

  1. First item
  2. Second item
  3. Third item

2. Unordered List (<ul>)

An unordered list is used when the sequence of items does not matter. It displays items with bullet points.

Example of Unordered List:

  • Item one
  • Item two
  • Item three

3. Definition List (<dl>)

A definition list is used to define terms and their corresponding definitions.

Example of Definition List:

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
